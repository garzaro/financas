#Projeto de finanças pessoais.

#Caso de uso

> A aplicação possui formulario de login<br/>
>Usuario poderá fazer cadastro de login<br/>
>Usuario poderá acessar a aplicação fazendo login<br/>
>Usuario estando logado poderá cadastrar outro usuário<br/>
>A aplicação é para controlar financas pessoais<br/>
>Usuario podera cadastrar lancamentos<br/>
>Lançamentos terão status como: PENDENTE, EFETIVADO E CANCELADO.<br/>
>Lancamentos terão tipos como: RECEITA E DESPESA<br/>
>
